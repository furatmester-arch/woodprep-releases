How it works

No CAD programming knowledge required

1

Install the plugin

Copy the DLL to your AutoCAD plugin folder. Runs on AutoCAD 2026.

2

Draw your cabinet

Use CORPUS_CREATION or draw from dimensions — the plugin builds the 3D model.

3

Run PP26 or EE26

One command generates the cutting list or full Biesse CNC program.

4

Send to production

Excel cutting list goes to the panel saw. BPP file goes straight to the machine.

Built by someone who uses it

Not a software company — a woodworking professional

I work in the woodworking industry and have been programming Biesse CNC machines for years. I built WoodPrep because AutoCAD is terrible at generating cutting lists — so I made it not terrible.

The plugin started as a personal tool. After a few hundred hours of development and real-world testing, it became something worth sharing. Every feature exists because I needed it on a real job.

This means you're not buying vaporware. WoodPrep is tested on real furniture production, real Biesse machines, and real AutoCAD drawings.
